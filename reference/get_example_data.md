# Get path to example data

Allows to access example data.

## Usage

``` r
get_example_data(name)
```

## Arguments

- name:

  Name of the file.

## Value

Path to example data.

## Examples

``` r
get_example_data("small_biocrates_example.xlsx")
#> [1] "/home/runner/.cache/R/renv/library/MetaboCrates-cfda6013/R-4.5/x86_64-pc-linux-gnu/MetaboCrates/extdata/small_biocrates_example.xlsx"
```
